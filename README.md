## ⚽ fulgorCupProject 
Pagina Web sobre el Mundial de futbol para el presidente de la FIFA Gianni Infantino.
Proyecto diseñado en Figma como prototipo de lo que será la página web a desarrollar utilizando diferentes herramientas.

## 🛠️ Stack Tecnológico
| Ecosistema | Tecnología / Herramienta | Uso en el Proyecto |
| :--- | :--- | :--- |
| **Frontend** | HTML5 & CSS3 | Estructura semántica del sitio y maquetación responsiva. |
| **Frontend** | JavaScript | Gestión dinámica de próximos partidos, estadísticas, clasificaciones y eliminatorias. |
| **Diseño** | Flexbox | Control total del diseño de layouts sin depender de frameworks externos. |
| **Datos** | JSON | Almacenamiento de la información de partidos, estadísticas y clasificaciones que consume JavaScript. |
| **Productividad** | Visual Studio Code | Editor de código principal, utilizando extensiones como Live Server para previsualización en tiempo real. |
| **Multimedia** | Figma | Diseño y edición de los assets visuales, banners y maquetas de presentación. |
| **Control de Versiones** | Git & GitHub | Gestión del historial de cambios y despliegue del repositorio. |
